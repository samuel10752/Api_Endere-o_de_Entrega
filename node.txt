Iniciar a instalacao dos modulos do node para o server.js funcionar:
npm install express
npm install soap
npm install cors

Para iniciar ele manualmente e so digitar esse codigo no terminal:
node server.js

Fazer que ele roda o server.js iniciar seu servidor automaticamente em qualquer sistema operacional e também mantê-lo em execução caso ele falhe:

Instale o pm2 globalmente usando o npm com o comando:
npm install pm2 -g

Inicie seu servidor com pm2 usando o comando:
pm2 start server.js

Configure o pm2 para iniciar automaticamente na inicialização com o comando:
pm2 startup

Salve a configuração atual do pm2 com o comando:
pm2 save
